Formalising Real Arguments #1

Published April 05, 2026

Someone put to me the following argument:

If businesses grow, they have to employ more people. By employing more people the economy improves. Businesses are not growing. Therefore the economy is not improving.

This is already very clear in its language, so translating into premise form is quite straightforward:

1. If businesses grow, they have to employ more people.
2. By employing more people the economy improves.
3. Businesses are not growing.
4. Therefore the economy is not improving.

The first thing to identify is an enthememe (an unspoken but implied premise) in the argument:

2a. If businesses grow, the economy improves.

So, in total, we have:

1. If businesses grow, they have to employ more people.
2. By employing more people the economy improves.
2a. If businesses grow, the economy improves.
3. Businesses are not growing.
4. Therefore the economy is not improving.

Now, we assign letters to the premises. Here is a sensible suggestion of one way we could assign letters:

G := “businesses GROW”
E := “they have to EMPLOY more people”
I := “the economy IMPROVES”

With this key at hand, we can now formulate the argument in Fitch-style as a natural deduction proof:

1. G -> E P
2. E -> I P
2a. G -> I (HS 1, 2)
3. ~G P
4. ~I (MT 2a, 3) X invalid [fallacy of denying the antecedent]

Modelled like this, the argument is invalid since it commits the fallacy known as the denying the antecedent. But this is just a mistake of modelling because the argument is really trying to say something more like this:

1’. Only if businesses grow do they have to employ more people.
2’. Only if they have to employ more people does the economy improve.
2a’. Only if businesses grow does the economy improve.
3’. Businesses are not growing.
4’. Therefore the economy is not improving.

Which, converted into a Fitch-style natural deduction proof, looks like this:

1. E -> G P
2. I -> E P
2a. I -> G (HS 1, 2)
3. ~G P
4. ~I (MT 2a, 3) valid

The distinction is subtle but important. By modelling the argument like this, the reasoning checks out as valid. This is likely why we take it to be a meaningful argument when it’s made so often, because it really does stand for valid reasoning after all.

From here though we can now examine each premise individually. Take 2a. It is a valid inference from 1 and 2, so if it doesn’t sound right then something is unsound about either 1 or 2. Let us see. 1 seems to check out empirically. 2 however is spurious. It has been challenged famously by Keynes who showed governments could make the economy grow artificially through fiscal stimulus even when businesses were not hiring.

So on these grounds, 2 can be rejected.

The last premise it is possible to challenge, seeing as this argument is valid, is 3. We may be able to pull up research suggesting that it isn’t the case that businesses aren’t growing. Perhaps this is a red herring. More complex economic forces are at work than the appearance we have suggests.
